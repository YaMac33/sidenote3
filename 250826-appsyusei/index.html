25.8.25 app.pyの必須修正ポイントマニュアル

app.py 必須の修正ポイント

1. タイムスタンプを日本時間にする

  • 現在 datetime.datetime.now().isoformat() を使っていますが、これはサーバーのローカルタイム（多くはUTC）です。
  • Python 標準の zoneinfo を使い、日本時間に変換します。

from zoneinfo import ZoneInfo

def save_room(title):
    room_id = str(uuid.uuid4())
    now = datetime.datetime.now(ZoneInfo("Asia/Tokyo")).isoformat()
    sheet_rooms.append_row([room_id, title, now, now])
    return room_id

def save_message(room_id, role, content_ja):
    msg_id = str(uuid.uuid4())
    now = datetime.datetime.now(ZoneInfo("Asia/Tokyo")).isoformat()
    sheet_msgs.append_row([msg_id, room_id, role, content_ja, "", now])
    return msg_id

2. チャットルームの名称変更・削除 API を追加


@app.route("/rooms/<room_id>", methods=["PATCH"])
def rename_room(room_id):
    new_title = request.json.get("title")
    rows = sheet_rooms.get_all_values()
    for i, r in enumerate(rows[1:], start=2):  # 2行目からデータ
        if r[0] == room_id:
            sheet_rooms.update_cell(i, 2, new_title)  # タイトルは2列目
            # 更新日時を日本時間で保存
            now = datetime.datetime.now(ZoneInfo("Asia/Tokyo")).isoformat()
            sheet_rooms.update_cell(i, 4, now)  # 更新日が4列目なら
            return jsonify({"status": "renamed"})
    return jsonify({"error": "room not found"}), 404

@app.route("/rooms/<room_id>", methods=["DELETE"])
def delete_room(room_id):
    rows_rooms = sheet_rooms.get_all_values()
    for i, r in enumerate(rows_rooms[1:], start=2):
        if r[0] == room_id:
            sheet_rooms.delete_rows(i)
            break

    rows_msgs = sheet_msgs.get_all_values()
    # 下から削除しないとインデックスがずれる
    for i, r in enumerate(reversed(rows_msgs[1:]), start=2):
        if r[1] == room_id:
            idx = len(rows_msgs) - i + 1
            sheet_msgs.delete_rows(idx)

    return jsonify({"status": "deleted"})

templates/index.html

  • サブメニュー（名前変更・削除ボタン）を追加する場所を特定したい
  • 回答メッセージのHTML構造をどう出力しているか確認したい

  • 「コードブロック単独コピー」機能をつけるために <pre><code> 構造を適切に変換する必要あり
  • 「太字などが反映されてしまう問題」は、<code> 内でエスケープ処理をすることで解決可能です。
  • Jinjaテンプレートで {{ content | e }} として出力する
  • もしくはサーバーで html.escape() を使ってエスケープ
