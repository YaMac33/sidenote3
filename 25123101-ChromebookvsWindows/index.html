Chromebook vs Windows
プログラミングを始めるならどっち？

「なるべく安く始めたい、でも失敗したくない。」
GitHubやGASを使った開発を検討中のあなたへ、賢いパソコンの選び方を解説します。

自分に合うPCを診断する

1. 基本的な違いを知る

まずはそれぞれのOSの得意・不得意を整理しましょう。Chromebookは「スマホ感覚」、Windowsは「万能型」と言えます。

スマホ感覚で使える軽快さ

  • 起動が爆速: 電源ボタンを押して数秒で作業開始。
  • セキュリティ: ウイルス対策ソフト不要。更新も自動。
  • 価格: 3〜5万円台で十分実用的なモデルが買える。
  • 弱点: Windows専用ソフト（.exe）は動かない。

GAS・Web開発との相性

Google Apps Script (GAS)はブラウザ上で動くため、Chromebookの得意分野です。 GitHub Codespacesを使えば、ブラウザの中にVS Code環境を作れます。

推奨ユーザー

Web開発入門者、サブ機が欲しい人

何でもできる万能選手

  • 汎用性: 仕事、ゲーム、編集、何でも対応可能。
  • ソフト: Office完全版、Adobeなど全てのPCソフトが動く。
  • 周辺機器: プリンター等の対応機器が圧倒的に多い。
  • 弱点: 更新やメンテナンスの手間が必要。安物は重い。

開発環境として

ローカルに環境を構築できるため、ネットがなくても開発可能。 ただし、快適に動かすにはそれなりのスペック（価格）が必要です。

推奨ユーザー

PCを本格的に学びたい人、ゲームもしたい人

2. なぜ安いChromebookで開発できる？

「GitHub Codespaces」を使えば、重い処理は全てインターネットの向こう側（クラウド）で行われます。 あなたのPCは、ただその結果を表示する「モニター」の役割をするだけです。

ポイント: GitHub Codespacesはブラウザの中に「VS Code」そのものを表示します。ChromebookでもWindowsでも、クラウドを使うなら処理速度は同じです。

3. 【重要】スペックの選び方

「安い端末でOK」と言っても、最低ラインがあります。特にメモリは快適さに直結します。

メモリ使用量のイメージ

※ブラウザで複数タブを開きながら開発する場合の目安

4GB は避けよう

OSとブラウザだけでメモリの大部分を使ってしまいます。調べ物のタブを開くとすぐに動作が重くなり、開発どころではありません。

推奨

8GB 以上を選ぼう

これなら余裕があります。CPUはコスパ最強の「Intel N100」搭載モデルが、3〜5万円台で見つかるため非常におすすめです。
