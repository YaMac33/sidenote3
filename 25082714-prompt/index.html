指定条件(プロンプト)に従ったWebページ生成機能の実装

指定されたプロンプトを使ってHTML化を行うように修正します。また、OpenAI APIのエラーも修正します。完璧です！指定されたプロンプトを実装しました。これで以下の機能が追加されました：

実装内容

  • カスタムプロンプト: 指定された6つの条件でHTML化
  • テキスト全文をマニュアル化
  • 本文の雰囲気に合わせたかっこいいデザイン
  • YY.MM.DD 件名形式のタイトル
  • 適切な件名の自動生成
  • プログラミングコードのコピー可能ブロック化
  • 商品・記事リンクの併記
  • OpenAI APIエラー修正: response_formatパラメータを削除してエラーを解消
  • 完全なHTML出力: ChatGPTが完全なHTMLドキュメントを生成するため、追加のラッピングが不要

使用方法

テキストボックスに本文を入力すると、ChatGPTが指定された条件に従って：

  • 適切なタイトル（日付 + 件名形式）
  • 構造化されたHTML
  • 美しいデザイン
  • コードブロックのコピー機能
  • リンクの自動抽出

などを含む高品質なWebページを生成します。

試しにテキストを入力してプレビューを確認してみてください！
