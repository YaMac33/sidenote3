GeminiとGoogleスプレッドシート連携

このマニュアルは、GeminiとGoogleスプレッドシートを連携させる様々な方法について、実際のQ&Aのやりとりを基に作成したものです。目的別に章立てされています。

第1章: Googleフォームの回答をGeminiで処理し、スプレッドシートに自動記録する方法

Q: Googleフォームから入力したプロンプトをGeminiで処理し、その結果をスプレッドシートに自動で書き込みたい。

Googleフォームの送信を「きっかけ」としてGoogle Apps Script（GAS）を動かし、Gemini APIと連携することで実現します。

設定手順

  1. フォームとスプレッドシートの準備：
    • Googleフォームを作成し、「回答」タブから回答の保存先となる新しいスプレッドシートを作成します。
    • スプレッドシートに「タイムスタンプ」「短文書き込みフォーム（プロンプト用）」列が自動生成されます。
    • Geminiの回答を書き込むための「長文書き込みフォーム（回答用）」列をC列に手動で追加します。
  2. Apps Scriptの設定：
    • スプレッドシートの「拡張機能」>「Apps Script」を開きます。
    • 下記のコードを貼り付け、`'ここに取得したAPIキーを入力してください'` の部分をご自身のAPIキーに書き換えます。
  3. トリガーの設定：
    • Apps Scriptエディタの「トリガー」（⏰）メニューから新しいトリガーを追加します。
    • 実行する関数を `onFormSubmit`、イベントの種類を `フォーム送信時` に設定して保存します。

GASコード

// ▼▼▼ あなたのGemini APIキーをここに貼り付けてください ▼▼▼
const GEMINI_API_KEY = 'ここに取得したAPIキーを入力してください';
// ▲▲▲ あなたのGemini APIキーをここに貼り付けてください ▲▲▲

const API_URL = `https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash-latest:generateContent?key=${GEMINI_API_KEY}`;

function onFormSubmit(e) {
  const sheet = SpreadsheetApp.getActiveSheet();
  const row = e.range.getRow();
  const userPrompt = e.values[1]; // フォームの最初の質問への回答

  if (!userPrompt || userPrompt.trim() === '') {
    return;
  }
  
  try {
    const geminiResponse = callGeminiAPI(userPrompt);
    sheet.getRange(row, 3).setValue(geminiResponse); // C列に回答を書き込む
  } catch (error) {
    sheet.getRange(row, 3).setValue(`エラー: ${error.message}`);
  }
}

function callGeminiAPI(prompt) {
  const payload = {"contents": [{"parts": [{"text": prompt}]}]};
  const options = {
    'method': 'post',
    'contentType': 'application/json',
    'payload': JSON.stringify(payload),
    'muteHttpExceptions': true
  };
  const response = UrlFetchApp.fetch(API_URL, options);
  const responseCode = response.getResponseCode();
  const responseBody = response.getContentText();

  if (responseCode === 200) {
    const json = JSON.parse(responseBody);
    return json.candidates[0].content.parts[0].text.trim();
  } else {
    throw new Error(`APIリクエスト失敗。ステータス: ${responseCode}, メッセージ: ${responseBody}`);
  }
}
ポイント：この方法は、すでにあるGoogleフォームとスプレッドシートの組み合わせでも問題なく動作します。

第2章: iPadでのGAS編集 トラブルシューティング

Q: iPadのブラウザでGASを編集したいのに、スプレッドシートのアプリが勝手に開いてしまう。

iPadOSのディープリンク機能により、リンクをタップすると関連アプリが自動で開いてしまいます。これを回避し、ブラウザで開くための方法がいくつかあります。

解決策1：リンクを長押しする（最も簡単）

  1. スプレッドシートへのリンクをタップするのではなく、指で長押しします。
  2. 表示されたメニューから「新規タブで開く」を選択します。

解決策2：「デスクトップ用Webサイトを表示」を使う

  1. SafariでGoogleドライブなどを開き、アドレスバーの左にある「ぁあ」アイコンをタップします。
  2. メニューから「デスクトップ用Webサイトを表示」を選択します。
  3. その状態で目的のスプレッドシートを開きます。
ポイント：ブラウザでデスクトップ表示にすることで、PCと同じように「拡張機能」メニューからApps Scriptエディタを開けるようになります。

第3章: スプレッドシートから直接Geminiを呼び出す方法

Q: Googleフォームを使わず、スプレッドシートのセルに書いたプロンプトで直接Geminiを実行したい。

スプレッドシートにカスタムメニューを追加し、選択したセルの内容をプロンプトとしてGeminiを呼び出すGASを組むことで実現します。

設定手順

  1. スプレッドシートのApps Scriptエディタに下記のコードを貼り付け、APIキーを設定して保存します。
  2. スプレッドシートの**画面を再読み込み**します。メニューバーに**「Geminiに依頼」**という項目が追加されます。

使い方

  1. B列にGeminiへのプロンプトを入力します。
  2. そのセルを選択した状態で、メニューの「Geminiに依頼」>「選択中のセルで実行」をクリックします。
  3. 初回実行時の承認後、A列にタイムスタンプ、C列にGeminiの回答が自動で書き込まれます。

GASコード

// (APIキーとAPI_URLの定義は第1章と同じ)
const GEMINI_API_KEY = 'ここに取得したAPIキーを入力してください';
const API_URL = `https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash-latest:generateContent?key=${GEMINI_API_KEY}`;


function onOpen() {
  SpreadsheetApp.getUi()
      .createMenu('Geminiに依頼')
      .addItem('選択中のセルで実行', 'callGeminiForSelectedCell')
      .addToUi();
}

function callGeminiForSelectedCell() {
  const ui = SpreadsheetApp.getUi();
  const sheet = SpreadsheetApp.getActiveSheet();
  const cell = sheet.getActiveCell();
  const row = cell.getRow();
  
  if (cell.getColumn() !== 2) {
    ui.alert('プロンプトをB列に入力し、そのセルを選択してから実行してください。');
    return;
  }
  
  const prompt = cell.getValue();
  if (!prompt || prompt.trim() === '') {
    ui.alert('選択されたセルにプロンプトが入力されていません。');
    return;
  }
  
  sheet.getRange(row, 3).setValue('Geminiが応答を生成中...');

  try {
    const geminiResponse = callGeminiAPI(prompt);
    sheet.getRange(row, 1).setValue(new Date()); // A列にタイムスタンプ
    sheet.getRange(row, 3).setValue(geminiResponse); // C列に回答
  } catch (error) {
    sheet.getRange(row, 3).setValue(`エラー: ${error.message}`);
    ui.alert(`エラーが発生しました: ${error.message}`);
  }
}

第4章: Geminiチャットルームの会話を保存する方法

Q: Geminiのチャット画面（gemini.google.com）でのやりとり自体を、自動でスプレッドシートに保存したい。

結論：現在、Geminiのチャットルームでの会話全体を**自動でスプレッドシートに保存する公式な機能はありません。**

チャット画面と開発者向けの「Gemini API」は別のサービスであり、連携する仕組みが提供されていないためです。しかし、代替方法がいくつか存在します。

方法1：Googleドキュメントにエクスポートする（公式機能）

スプレッドシートではありませんが、会話全体をGoogleドキュメントとして保存する公式機能が最も便利です。

  1. Geminiの回答の下にある共有アイコン（上向き矢印の付いた箱）をクリックします。
  2. 「Googleドキュメントにエクスポート」を選択します。

方法2：手動でコピー＆ペーストする

最もシンプルで、好きな部分だけを保存できる確実な方法です。Geminiとの会話内容をマウスで選択してコピーし、スプレッドシートのセルに貼り付けます。

方法3：ブラウザの拡張機能を利用する（非公式）

Chromeウェブストアなどで、表示中のページ情報をスプレッドシートに送る拡張機能を探す方法です。ただし、**セキュリティリスク**が伴うため、信頼できる提供元かどうかの確認が必須です。
